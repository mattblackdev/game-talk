# 3D Game Development with React Three Fiber

Let there be light!

```
git clone https://github.com/mattblackdev/game-talk.git
cd game-talk
npm i
```

Update `.env.local` > `VITE_REPO_PATH` to yours

```
npm start
```

- Lights, Camera, Canvas
- 3d coordinates x,y,z
- Sky shader + light
- Ground (Drei)
- Models (GLTF)
- Player input
- Interaction
- Animations
- Particles
- Gameloop
- Physics
- Dialog
- Water
- HUD
